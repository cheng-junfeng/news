# news

### 1.关于模块的说明

- 划分为了4个模块，可以根据需要裁剪。具体如下：
- library:  	   基准库    （抽取了公用部分）
- hintlib:         提示模块   (弹框/加载条/Toast)
- customlib:       自定义布局模块   (左滑删除/编辑全清/倒计时等)
- hellocharts:     图表模块   (基于hellocharts)
